# Investment Platform UI

## Project Description

Create a modern and professional investment platform frontend using Next.js, React, TypeScript, and Tailwind CSS.

This project is frontend only and should be structured for future backend integration.

### Required Pages

- Login
- Dashboard
- About Project
- Investment Information
- Wallet
- Deposit
- Withdraw
- Transaction History
- User Profile
- Settings
- Support Center

### UI Requirements

- Modern design
- Responsive layout
- Sidebar navigation
- Top navigation bar
- Dark mode support
- Reusable components
- Clean folder structure
- Professional animations
- Mobile friendly

### Wallet

The wallet page should contain:

- Current Balance
- Deposit Button
- Withdraw Button
- Transaction History

The Deposit and Withdraw buttons are UI only and will be connected to backend APIs later.

### Future Integration

The project must be organized so it can later connect with:

- Authentication API
- Database
- Wallet API
- Admin Panel
- Support System

### Technologies

- Next.js
- React
- TypeScript
- Tailwind CSS
- React Icons

### Goal

Build a clean, scalable, and production-ready frontend that can later be connected to backend services without changing the UI structure.
